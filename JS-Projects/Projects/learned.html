Tips I've Learned About Learning to Code

  1. Be Patient. There is a lot to learn. It's going to take time.
  2. Approach everything with the confidence that eventually I will be able to teach this to others.
  3. Maintain a balance of spending time between Training Materials, YouTube Tutorials and Hands-on Practice.
  4. When I get stuck, take a break and come back.
  5. If I can't figure something out, go through this list in order:
    • DevTools
    • Console.log
    • ChatGPT
    • Source Websites
    • Google Searching
    • YouTube Searching
    • Post a Question on Message Boards
  6. Read a lot of other people's code and look up everything I can't explain.
  7. The first step in creating a successful project is breaking it down into smaller parts. And sometimes breaking those parts down even more.
  8. When something doesn't work, often it's because I'm skipping a step. Read through the code again and make sure I'm not trying to do two things at once.
  9. Take time to enjoy what I create, and also see if I can 'break' it as a user.
